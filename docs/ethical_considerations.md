# Ethical Considerations

The software must not infer nationality, religion, ethnicity, or real-person identity from ambiguous appearance. Group assignment should rely on explicit symbols, readable text, configured reference material, or manual review and must include uncertainty.

Remote API adapters are opt-in. Local images are not uploaded unless a remote adapter is explicitly configured and enabled. Automated political interpretations should be reported as model measurements and accompanied by evidence, uncertainty, alternative interpretations, and validation results.
